# 📚 Topics Covered

## Leap Year Program

Learned how to determine whether a given year is a leap year using conditional statements.

### Logic

- If the year is divisible by 400 → Leap Year
- Else if divisible by 4 but not by 100 → Leap Year
- Otherwise → Not a Leap Year
